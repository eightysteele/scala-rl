How can I help?

The ScalaRL community welcomes and encourages contributions! Here are a few ways to help out:

  • Look at the code coverage report, find some untested code, and write a test for it. Even simple helper methods and syntax enrichment should be tested.
  • Find an open issue, leave a comment on it to let people know you are working on it, and submit a pull request.

See the contributing guide for more information.
